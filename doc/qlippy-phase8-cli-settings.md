# Phase 8 — CLI mode and settings

## CLI mode

### Argument parser

```text
--daemon
--popup
--toggle
--copy id
--delete id
--clear
```

### If popup mode

```text
connect IPC
send command
exit
```

### If daemon not running

```text
start daemon
retry IPC
```
